您的每一笔质押、每一次收益领取，都严格按照链上代码中写定的规则执行，并受到 CertiK 审计的持续保障。

## 五、资金安全——没有人能动您的钱

这是用户最关心的问题，我们给出最明确的回答：

**合约中不存在任何允许管理员转走用户资金的功能。**

- 您的资金存储在区块链上的专用账户（PDA）中，由合约程序控制
- 所有资金操作（质押、赎回、领取收益）都必须由您本人的钱包签名

即使 VELA 的网站关闭，您的资金依然安全地存在于区块链上，随时可以通过链上交互取回。详细的资金取回方式请参阅《资金安全说明》。

## 七、不依赖网站——真正的去中心化

VELA 是一个完全部署在 Solana 区块链上的去中心化协议。网站只是一个方便您操作的界面，合约本身独立运行在区块链上。

我们提供了开源的交互脚本，即使不通过网站，您也可以直接与合约交互完成所有操作。脚本代码位于项目仓库的 `scripts/stake-example.js`，使用方式如下：

1. 安装 Node.js 环境（v16 或以上版本）
2. 克隆代码仓库并安装依赖
   ```bash
   git clone https://github.com/veladex/vela-anchor.git
   cd vela-anchor/scripts
   npm install
   ```
3. 在 `stake-example.js` 中配置您的信息：
   - 将 `privateKeyString` 替换为您的钱包私钥（bs58 格式）
   - 将 `connection` 的 RPC 地址改为 Solana 主网 RPC（如 `https://api.mainnet-beta.solana.com`）
   - 将 `mintAddress` 改为主网 VELA 代币地址
4. 根据需要调用对应的函数：
   ```bash
   node stake-example.js
   ```

脚本中包含了完整的操作示例：

| 函数名 | 功能 | 说明 |
|--------|------|------|
| `addReferral` | 绑定推荐人 | 质押前必须先绑定推荐人 |
| `createStake` | 创建质押 | 指定金额和周期类型（1=7天, 2=30天, 3=90天） |
| `getMyStakingOrders` | 查询质押订单 | 查看所有进行中的质押 |
| `claimInterest` | 领取利息 | 领取指定订单的累积利息 |
| `unstake` | 解质押 | 到期后赎回本金和剩余利息 |

所有操作只需要您自己的钱包签名，不需要任何第三方授权。

> 注意：如果您不熟悉技术操作，可以请社区中任何开发者帮您验证和执行——因为代码是完全开源的，任何人都可以确认脚本的安全性。

## 九、常见疑问

**Q: 项目方会不会跑路带走资金？**
A: 不会。合约中没有任何管理员提款功能，资金只能由用户本人通过钱包签名操作。即使团队解散，您的资金仍然安全地在链上，随时可以取回。

**Q: 合约会不会被偷偷升级加入后门？**
A: 不会。合约代码已通过 CertiK 审计并生成唯一的程序哈希值。任何代码修改都会改变哈希值，无法通过审计验证；即使用源码重新部署也会产生新的程序地址，同样不会被 CertiK 认可。您可以随时通过 `solana-verify` 工具独立验证链上代码与开源仓库的一致性。

**Q: 利率会不会被暗中修改？**
A: 不会。利率参数写死在合约代码中，不是可修改的变量。要改利率就必须部署全新的合约（新的地址），旧合约中的质押不受任何影响。

**Q: 我的推荐收益计算公平吗？**
A: 推荐收益的计算逻辑完全写在链上合约中，对所有人一视同仁。您可以查看开源代码或审计报告来验证计算规则。

**Q: NFT 会不会无限增发稀释价值？**
A: 不会。Diamond NFT 上限 600 枚、Gold NFT 上限 12,000 枚，这些数字硬编码在合约中，无法修改。

**Q: 如果我不懂技术，怎么确认这些说法是真的？**
A: 您可以请任何懂 Solana 开发的朋友帮您验证，或者参考 CertiK 的审计报告。我们的代码完全开源，经得起任何人的检验。

**Q: 合约会不会被黑客攻击？**
A: VELA 合约经过世界排名第一的 CertiK 专业审计，代码使用 Solana 生态最成熟的 Anchor 框架开发，遵循行业最高安全标准。同时代码完全开源，持续接受全球安全研究者的审视。

**Q: 我的私钥会不会被网站窃取？**
A: 不会。VELA 合约和网站不会存储或接触您的私钥。所有交易签名都在您的钱包（如 Phantom）本地完成。前端代码也已开源，任何人都可以检查确认。请务必妥善保管您的私钥或助记词，这是您资金安全的唯一保障。
